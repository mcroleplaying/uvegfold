## 1. Szülőföld

Dobj d100-zal és nézd meg az alábbi táblázatban, Kherret melyik vidékén született a karaktered, vagy válassz szülőföldet egyenesen a táblázatból!

| d100 | Szülőföld |
| ---- | --------- |
| 1–20 | Észak-Kherret |
| 21–40 | Dél-Kherret |
| 41–50 | Közép-Kherret |
| 51–57 | Hegyvidék |
| 58–65 | Erdőség |
| 66–80 | Tengerpart |
| 81–95 | Szigetvilág |
| 96–100 | Nyugati Part |
